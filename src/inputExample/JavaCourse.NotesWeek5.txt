[definition]The Java Class Loader - part of the Java Runtime Environment, dynamically loads Java classes into the Java Virtual Machine.[1] Usually classes are only loaded on demand. The virtual machine will only load the class files required for executing the program. The Java run time system does not need to know about files and file systems as this is delegated to the class loader.
